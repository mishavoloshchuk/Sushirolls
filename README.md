# About SushiRolls
**SushiRolls** - is a online store with **responsive design** and **themes**, user **authorization**, role-based **access control**.
### Technology stack 
- HTML
- CSS
- JavaScript
  - REST API
  - AJAX
- PHP
  - MySQL database
